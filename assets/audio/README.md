# Meditation Audio Assets

This folder should contain meditation bell/chime sounds for the meditation feature.

## Required Files

Add one of these files to enable meditation chimes:

- `meditation_bell.mp3` - MP3 format (recommended)
- `meditation_bell.wav` - WAV format (alternative)

## Where to Find Free Meditation Bell Sounds

1. **Freesound.org** - Search for "meditation bell" or "singing bowl"
   - Many sounds under Creative Commons licenses
   - Example: https://freesound.org/search/?q=meditation+bell

2. **ZenMix** - Free meditation sounds
   - https://www.zenmix.com/

3. **Pixabay** - Free sound effects
   - https://pixabay.com/sound-effects/search/meditation%20bell/

## Recommended Sound Characteristics

- Duration: 2-5 seconds
- Type: Single bell/bowl strike
- Quality: Clean, peaceful tone
- Format: MP3 (smaller file size) or WAV (higher quality)
